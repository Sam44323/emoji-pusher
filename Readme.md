# **Scripts**

**clone the repo**

```bash
git clone https://github.com/Sam44323/emoji-pusher.git
```

---

**Install the dependencies**

```bash
yarn
```

**Running the emulator**

```bash
yarn start
```

**Run the following for devices**

#### `i: ` for ios

#### `a: ` for android

#### `w: ` for web
